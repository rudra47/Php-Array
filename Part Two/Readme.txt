Functions: 

json_decode() -> Convert into php object from the json object.
array_walk() -> Run each array element in a user defined function.
array_walk_recursive() -> Same as array_walk(). Difference is this function can work with depper array (an array inside an array).
array_splice() -> It replace or add with new array element. It also return an array. 
ex: array_splice($a1, 2, 0, $a2) = add
    array_splice($a1, 0, 2, $a2) = replace

sort()   -> sort arrays in ascending order
rsort()  -> sort arrays in descending order
asort()  -> sort associative arrays in ascending order, according to the value
ksort()  -> sort associative arrays in ascending order, according to the key
arsort() -> sort associative arrays in descending order, according to the value
krsort() -> sort associative arrays in descending order, according to the key

explode() -> The explode() function breaks a string into an array.

array_unshift() -> The array_unshift() function inserts new elements to an array. Values will be inserted in the beginning of the array. 
array_fill() -> The array_fill() function fills an array with values.
                Syntax: array_fill(index, number, value)
